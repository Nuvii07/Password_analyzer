# Password_analyzer
To run script main.py

```python main.py```

## General information
This script has two password validators:

- basic password validator:
  - validate if password has mor then 8 characters,
  - validate if password has lowercase and uppercase letters,
  - validate if password has any special character.

- haveibeenpwned.com api based validation:
  * is a website that allows Internet users to check whether their personal data has been compromised by data breaches.

If you want to validate batch passwords write them to check_passwords.txt file,
if password is safe, it will be written to safe_passwords.txt file.

## Technologies used
- Python 3.8.3
- Api
- Pytest
